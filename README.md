# Japan Trip Website

## How to run locally

1. Open this folder in Visual Studio Code.
2. Open Terminal.
3. Run:

```bash
npm install
npm run dev
```

4. Open the local link shown in the terminal, usually:

```txt
http://localhost:5173
```

## How to host

You can upload this project to GitHub, then deploy using Vercel.

Vercel settings:
- Framework: Vite
- Build command: npm run build
- Output directory: dist
